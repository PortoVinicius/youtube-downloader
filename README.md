# youtube-downloader

Estrutura:
```
youtube-downloader
│
├── venv
├── downloads
├── templates
│   └── index.html
│
└── app.py
```
## Criar ambiente virtual
```bash
python -m venv venv
```

## Ativar o ambiente virtual
```bash
venv\Scripts\activate

source venv/bin/activate
```

## Instalar as bibliotecas
```bash
pip install flask yt-dlp
```

## Rodar o programa
```
python app.py
````
